# OIBSIP
Task1:Online Reservation System
🚆 A Java-based Online Reservation System that allows users to log in, book train tickets, generate PNR numbers, and manage reservations through a simple menu-driven interface.
🎫 It supports booking, cancellation, and viewing of tickets with passenger details, train info, and automatically generated seat numbers.
🔐 The system includes authentication and stores reservations using a HashMap for quick retrieval and management.
Task2:Number Guessing Game
🎮 A Java-based Number Guessing Game called “Mystery Vault” where players try to unlock a secret number between 1 and 100 within limited attempts.
🏆 The game includes multiple rounds, a scoring system based on performance, and helpful hints like “Too High,” “Too Low,” and “Very Close.”
📊 At the end, players receive a final score and rank based on their average performance across all rounds.
